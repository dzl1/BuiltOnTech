What are Fractions?

A fraction represents a part of a whole. It has a numerator (top number) and a denominator (bottom number).

How to Add Fractions

To add fractions with the same denominator, add the numerators and keep the denominator the same. For different denominators, find a common denominator first.

  • 1/4 + 2/4 = 3/4
  • 1/3 + 1/6 = 1/3 + 2/6 = 2/6 + 1/6 = 3/6 = 1/2
