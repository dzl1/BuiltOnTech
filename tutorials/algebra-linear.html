What is a Linear Equation?

A linear equation is an equation that makes a straight line when it is graphed. It has the form ax + b = c.

How to Solve Linear Equations

  1. Isolate the variable (x) on one side of the equation.
  2. Perform the same operation on both sides to keep the equation balanced.
  3. Example: x + 3 = 7 → x = 7 - 3 → x = 4

Practice Problems

  1. x + 5 = 12
  2. 2x = 10
  3. x - 4 = 3
